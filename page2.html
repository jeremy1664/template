UNIT 1, OUTCOME 3 (REVIEWS)

Review 3.1

Review the benefits and drawbacks of IT tools and systems used in terms of productivity and efficiency – could you have been more efficient when making your website? How? Could you have done it in teams?

Xcode is very buggy so it can cause as many problems as it solves, especially whem yoiu are new, inexperienced and don't understand the difference between me making a mistake and the program having a bug...if I had experience I would know my programming was correct and deduce that there was a problem elsewhere but I'm not at that stage yet! One thing that I have learned is that all programs have their own "formats" so a photo or gif that runs in one program will not always work on another so eiother need to select another image with a different "file type" ie .jpg, .gif, .png etc to make them compatible. With my lack of experience I don't think I could have been more efficient while making my website, but if I could-I think if I only focussed on doing 100% of the contenmt first, and work on the styling afterwards, perhaps this would be more efficient? Teams would have helped do it quickly and efficiently but then I wouldn't know how to do it myself from scratch so it would be beneficial to the webpage being created but detrimental to my understanding of how its made.

Review 3.2

Describe ways to improve productivity and efficiency

Work in "chunks", design all content ideas and have all the content ready to go, file all images needed, so you can complete task after task, one by one-until completion. put time constraints on each task so that ideas got completed ahead of time, on time or behind schedule so that you can keep focus and not drift off. Upload the project onto github so that you can work remotely at anytime. The best way to increase efficiency is to increase training, knowledge and experience working with these tools!

Develop solutions to improve own productivity using IT in digital projects – what would you do differently next time? Team work? Use of Slack?

I feel it is organisation that is key to improved productivity and that for the purposes of this task, and for me it's actually ignoring the IT elements, and simply typing out all the content so that it is ready for IT styling etc afterwards. teamwork clearly would've speeded up this task but it's irrelevant here as this project was a solo effort.
